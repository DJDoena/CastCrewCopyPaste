General Info

This plugin allows you to easily copy and paste cast and crew data without actually opening the profile for editing.
While pasting it also avoids a bug in the built-in paste when it comes to the "puppeteer" information for a cast member.

You also have the option to use this plugin as a data receiver for the Cast/Crew Edit 2 (version 2.1.8.0 and higher) tool.
You do this be enabling the option in DVD Profiler in the Tools menu as well as in Cast/Crew Edit 2 in the Settings>Other menu.
